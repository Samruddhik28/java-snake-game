#  Java Snake Game 2026
A smooth, interactive, and retro-style Snake game built using **Java Swing** and **AWT**.

## 🎮 Gameplay Overview
This project is a modern take on the classic Snake game. It features a responsive GUI, real-time score tracking, and a clean codebase designed for educational purposes.

### 🕹️ Controls
* **Arrow Keys**: Move the snake (Up, Down, Left, Right).
* **Spacebar**: Restart the game after a 'Game Over'.
* **ESC**: Exit the application.
